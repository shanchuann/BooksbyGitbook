# 书籍介绍

当我读完一些关于C语言的书籍或是后，发现他们要么过于全面，要么过于混乱，对于新手来说这并不是一件好事。所以这本书就因此而来——它可以帮助你系统化建立有关C语言的知识体系，甚至足够你后续对技术的更深一层次的探索。

这本书有一个配套的[CStudy](https://github.com/shanchuann/CStudy.git)练习系统，随着本书的更新它里面的习题也会逐步更新。
